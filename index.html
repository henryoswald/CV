I have only achieved this because I love programing and building things, I live inside Eclipse and Vim with Java is my bread and butter. My main web development experience has been using the impressive Java Play! Framework, which exposed me to things like JUnit, Hibernate and the lovely world of MVC, but I still have a lot to learn.

Perhaps the greatest surprise on this course has been the experiences of working in largely international teams on projects. I chose these technologies because they are all very different to my current core strengths, Java and Relational Databases, but also would just like to learn and understand them.

I am happiest in UNIX machine and have found Mecurial to be my preferred VCS. I poke around everything I can when I have the time, from Amazon Web Services to the canvas element in HTML5 to SASS and writing little scripts in Python, I don't claim to be a master in any of these, but I am interested and always looking to learn more.
